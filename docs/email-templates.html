Supabase 변수 참고

  • {{ .ConfirmationURL }} — 인증/로그인/재설정 링크 (1·2·3·4번에서 사용)
  • {{ .Token }} — 6자리 OTP 코드 (5번)
  • {{ .NewEmail }} — 변경할 새 이메일 (4번)
  • {{ .Email }} / {{ .SiteURL }} / {{ .RedirectTo }} — 필요 시 추가 활용
